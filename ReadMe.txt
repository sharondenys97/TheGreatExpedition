Hello, this is a user guide on how to run The Great Expedition

The source folder is named Great Expedition V1
If you would like to run and play the game you need the GreatExpeditionWindows file that contains the executable file.

How to run?

1. Download GreatExpeditionWindows
2. Unzip the file and extract the resources(important).
3. Look for the exe file “Great Expedition V1.exe”
4. Click on it
5. A window will pop up, choose the size and quality of the screen according to your preference

6. Make sure to check the windowed box to make it easier to exit.

This game was developed on Mac and build for Windows. It should be able to run as compatibility testing was done.

Thank you and enjoy!
